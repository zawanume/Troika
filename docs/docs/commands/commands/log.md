# `システム情報`コマンド
ホストされているサーバーやプロセスに関する技術的な情報を表示します。引数(`mem`、`cpu`、`basic`のうちいずれか)を指定して特定の内容のみ取得することもできます。

## エイリアス
- ログ
- log
- システム情報
- systeminfo
- sysinfo

## 使い方
```
sysinfo [mem|cpu]
```

## 使用例
```
sysinfo mem
```


## 実行に必要な権限
なし

※管理者権限や、サーバーの管理権限を持つユーザーはこの権限を満たしていなくてもいつでもこのコマンドを実行できます。
